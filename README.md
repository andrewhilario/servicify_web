# Servicify

Servicify is a service platform.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```bash
python -m venv env
source env/Scripts/activate
pip install django
```

## Usage

```python
python manage.py runserver
```
